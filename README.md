# Product Store Website

This is my HTML assignment created as part of my Web and App Development learning journey at SMIT.

## About The Project

Product Store is a simple multi-page website built using HTML. 
The project contains a home page, shop page, and different product category pages.

The purpose of this assignment was to practice HTML page linking, folder structure, navigation, and organizing website content.

## Technologies Used

- HTML5

## Website Structure

- Home Page
- Shop Page
- Fragrance Category
- General Items Category
- Groceries Category
- Medical Category

## Features

- Multi-page website structure
- Navigation between pages
- Product categories using lists
- Internal page links
- Organized folder structure

## What I Learned

Through this assignment, I learned:

- How to create multiple HTML pages
- How to connect pages using anchor tags
- How relative paths work
- How to organize website files and folders
- How to create a basic website navigation system
